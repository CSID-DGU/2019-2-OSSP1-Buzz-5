# 2019-2-OSSP1-Buzz-5
WebRTC video course app with ReactJS

## 프로젝트 설명
YouTube 강의 영상을 많은 사람들이 시청하고 배우는 것에서 아이디어를 착안하여 간단한 회원가입을 통하여 누구나 튜터, 튜티가 될 수 있는 인터넷 강의 커뮤니티를 만들고자 하였습니다.
